# Game Shop App

This repository contains a full-stack **Game Shop Application** developed as a mini project for the course **1204208 - Advanced Web Development and Management** in the **Computer Science Program, Mahasarakham University**.

The project consists of:
- a **backend** developed with Node.js and TypeScript,
- a **frontend** developed with Angular,
- and a **MySQL database** managed through Docker Compose.

## Project Structure

- `backend/` - REST API server built with Node.js and TypeScript
- `frontend/` - Angular-based client application
- `docker-compose.yml` - local development environment configuration
- `init.sql` - database initialization script

## Technologies

- Angular
- Node.js
- TypeScript
- MySQL
- Docker
# Project Objective

The objective of this project is to apply the knowledge and concepts learned in the course to develop a full-stack mini project.
